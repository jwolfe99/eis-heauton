# Eis Heauton
Just enough styling to make Roam pop.
- Light and Dark modes
- Better typography defaults
- Prettier Kanban and Tables
- Wider search

How to add this theme:
```
@import url("https://sushaantu.github.io/eis-heauton/style.css");
```
